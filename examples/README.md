# Examples

Examples are split by language:

- [`python/`](./python/) for Python SDK and Python agent-framework examples
- [`typescript/`](./typescript/) for TypeScript SDK and TypeScript agent-framework examples

The canonical flow is the same in both languages:

1. use a paired wallet and API key
2. submit a pact
3. wait until the pact is active
4. execute a blockchain action
5. handle denial feedback and retry with compliant parameters

Recipe examples are intentionally not included here. Recipe discovery and distribution now belong to the CAW recipe library in the product and website surfaces.
